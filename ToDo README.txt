README
1. Download and install WAMPServer from www.wampserver.com
2. Navigate to wamp64 folder on harddrive and go to www folder
3. Create a new folder named ToDoList
4. Login to server page by going to localhost on your web browser
5. Connect to phpmyadmin with username: root and password: ''
6. Create database called todolist
7. Create new table called tasks and add 3 columns
8. Name columns TaskNum, TaskName, and TaskDes
9. Auto-increment TaskNum
10. Copy Index.html file and Todo.php file into ToDoList project folder inside wamp64/www/ToDoList folder
11. Go to localhost page and navigate to ToDoList Project folder.
12. Add a new task, view existing tasks, or delete a task from list.
